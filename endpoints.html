user_account endpoints

GET /
POST /api/login

Example request:

{
  "username": "admin",
  "pwd": "1234",
  "remember": true
}

Example response:

{
    "data": {
      "token": "user_token"
    },
    "msg": "success",
    "status": "OK"
}
Status Codes:

  • 200 OK – no error

POST /api/registration

Example request:

{
  "username": "Admin",
  "full_name": "Admin A.A.",
  "email": "t@t.t",
  "phone": null,
  "contacts": null,
  "pwd": "1234"
}

Example response:

{
  "status": "OK",
   "msg": "Check your email...",
}

{
  "status": "ERR",
   "msg": "username is not allowed.",
}
Status Codes:

  • 200 OK – no error
