user_account endpoints

Login

POST /user_account/login

Adds auth cookies if login successful

Example request:

{
  "username": "admin",
  "pwd": "1234",
}

Example response:

{
    "msg": "login successful",
    "token": "user_token"
}
Status Codes:

  • 200 OK – no error

  • 401 Unauthorized – Wrong username or password

Registration

POST /user_account/registration

Example request:

{
  "username": "Admin",
  "full_name": "Admin A.A.",
  "email": "t@t.t",
  "phone": null,
  "contacts": null,
  "pwd": "1234"
}

Example response:

{
   "msg": "Check your email...",
}
Status Codes:

  • 200 OK – no error

  • 400 Bad Request – username is not allowed

Reset password request

POST /user_account/reset_password_request

[requires NO auth cookies or headers]

Example request:

{
  "username": "Admin"
}

Example response:

{
   "msg": "Check your email...",
}
Status Codes:

  • 200 OK – no error

  • 400 Bad Request – user not found

  • 403 Forbidden – user is already authenticated

Profile settings

GET /user_account/profile_settings

[requires auth cookies or headers]

Example response:

{
   "contacts": null,
   "email": "d******z@yandex.ru",
   "full_name": "Admin A.A.",
   "is_email_verified": false,
   "phone": null,
   "pwd": "********",
   "username": "Admin"
}
Status Codes:

  • 200 OK – no error

  • 401 Unauthorized – Missing JWT in headers or cookies (Missing Authorization Header; Missing cookie “access_token_cookie”)

  • 403 Forbidden – Token has expired
